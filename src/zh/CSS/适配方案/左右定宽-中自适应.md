## flex

```less {5,6,11,15,19}
.left-right-fixed-width-center-self-adaption-flex-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  display: flex;
  flex-direction: row;
  & > div {
    height: 100%;
  }
  .left {
    width: 100px;
    background-color: cornflowerblue;
  }
  .right {
    width: 100px;
    background-color: cornflowerblue;
  }
  .center {
    flex: 1 1 0;
    background-color: greenyellow;
  }
}
```

## float

### float + margin(有注意事项)

```less {9,10,14,18,19}
.left-right-fixed-width-center-self-adaption-float-margin-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  & > div {
    height: 100%;
  }
  .left {
    width: 100px;
    float: left;
    background-color: cornflowerblue;
  }
  .center {
    margin: 0 100px;
    background-color: greenyellow;
  }
  .right {
    width: 100px;
    float: right;
    background-color: cornflowerblue;
  }
}
```

### float + calc

```less {7,10,14,18}
.left-right-fixed-width-center-self-adaption-float-calc-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
```
