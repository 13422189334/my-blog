> `focus-within`: 当前元素以及后代元素的聚焦事件 

::: vue-playground
@file App.vue
```vue
<template>
  <div class="focus-within-demo">
    <div class="line">
      <label class="label">
        <span>姓名</span>
        <input class="input">
      </label>
    </div>
    <div class="line">
      <label class="label">
        <span>身份证号</span>
        <input class="input">
      </label>
    </div>
  </div>
</template>
<style scoped>
.focus-within-demo {
  width: 100%;
  input:focus {
    border-color: #409eff;
  }
  .line {
    transition: 0.5s;
    &:focus-within {
      background-color: #eee;
    }
  }
}
</style>
```
:::
