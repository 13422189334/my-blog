## flex

```less {5,6,11,15}
.left-fixed-width-right-self-adaption-flex-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  display: flex;
  flex-direction: row;
  & > div {
    height: 100%;
  }
  .left {
    width: 100px;
    background-color: cornflowerblue;
  }
  .right {
    flex: 1 1 0;
    background-color: greenyellow;
  }
}
```

::: vue-playground flex demo
@file App.vue
```vue
<template>
  <div class="left-fixed-width-right-self-adaption-flex-demo">
    <div class="left">left</div>
    <div class="right">right</div>
  </div>
</template>
<style scoped>
.left-fixed-width-right-self-adaption-flex-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  display: flex;
  flex-direction: row;
  & > div {
    height: 100%;
  }
  .left {
    width: 100px;
    background-color: cornflowerblue;
  }
  .right {
    flex: 1 1 0;
    background-color: greenyellow;
  }
}
</style>
```

### float + calc
