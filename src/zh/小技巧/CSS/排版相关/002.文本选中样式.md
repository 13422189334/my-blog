> `selection`：文本选中样式选择器

::: normal-demo 演示
```html
<div class="selection-demo">
  this is something text;this is something text;this is something text;
  this is something text;this is something text;this is something text;
  this is something text;this is something text;this is something text;
</div>
```

```css {4,7}
.selection-demo {
  width: 100%;
}
.selection-demo::selection {
   background-color: #333;
   color: #fff;
}
```
:::
