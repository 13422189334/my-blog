> `first-letter`: 当前元素的第一个字母

::: vue-playground
@file App.vue
```vue
<template>
  <div class="first-letter-demo">
    this is something text;this is something text;this is something text;
    this is something text;this is something text;this is something text;
    this is something text;this is something text;this is something text;
  </div>
</template>
<style scoped>
.first-letter-demo {
  width: 100%;
  &:first-letter {
    font-size: 2em;
    text-transform: uppercase;
    float: left;
    line-height: 1;
    margin-right: 10px;
  }
}
</style>
```
:::
