> `has`: 表示满足一定条件后，就会匹配该元素

::: vue-playground
@file App.vue
```vue
<template>
  <div class="has-data-required-demo">
    <div class="line">
      <label class="label">
        <span>姓名</span>
        <input data-required class="input">
      </label>
    </div>
    <div class="line">
      <label class="label">
        <span>身份证号</span>
        <input class="input">
      </label>
    </div>
  </div>
</template>
<style scoped>
.has-data-required-demo {
  width: 100%;
  .label {
    span:has(+ input[data-required])::after {
      content: '*';
      color: red;
    }
  }
}
</style>
```
:::
