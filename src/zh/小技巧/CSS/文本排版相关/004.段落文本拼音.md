> `pinyin.js` 用于生成拼音， `<ruby><rt></rt></ruby>` 用于渲染元素

::: normal-demo 演示
```html
<div class="textLayout-004-demo">
  <ruby>段<rt>duàn</rt></ruby>
  <ruby>落<rt>luò</rt></ruby>
  <ruby>文<rt>wén</rt></ruby>
  <ruby>本<rt>běn</rt></ruby>
  <ruby>拼<rt>pīn</rt></ruby>
  <ruby>音<rt>yīn</rt></ruby>
</div>
```
:::
