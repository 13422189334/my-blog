> `first-letter`: 当前元素的第一个字母

::: normal-demo 演示

```html
<div class="first-letter-demo">
  this is something text;this is something text;this is something text;
  this is something text;this is something text;this is something text;
  this is something text;this is something text;this is something text;
</div>
```

```css {4,10}
.first-letter-demo {
  width: 100%;
}
.first-letter-demo:first-letter {
   font-size: 2em;
   text-transform: uppercase;
   float: left;
   line-height: 1;
   margin-right: 10px;
 }
```
:::
