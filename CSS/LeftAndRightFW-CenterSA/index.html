flex

.left-right-fixed-width-center-self-adaption-flex-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  display: flex;
  flex-direction: row;
  & > div {
    height: 100%;
  }
  .left {
    width: 100px;
    background-color: cornflowerblue;
  }
  .right {
    width: 100px;
    background-color: cornflowerblue;
  }
  .center {
    flex: 1 1 0;
    background-color: greenyellow;
  }
}

float + margin(有注意事项)

.left-right-fixed-width-center-self-adaption-float-margin-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  & > div {
    height: 100%;
  }
  .left {
    width: 100px;
    float: left;
    background-color: cornflowerblue;
  }
  .center {
    margin: 0 100px;
    background-color: greenyellow;
  }
  .right {
    width: 100px;
    float: right;
    background-color: cornflowerblue;
  }
}








 
 



 



 
 



float + margin demo 需调整 中间元素 和 右边元素 位置

float + calc

.left-right-fixed-width-center-self-adaption-float-calc-demo {
  width: 100%;
  height: 250px;
  border: 1px solid;
  & > div {
    height: 100%;
    float: left;
  }
  .left {
    width: 100px;
    background-color: cornflowerblue;
  }
  .right {
    width: 100px;
    background-color: cornflowerblue;
  }
  .center {
    width: calc(100% - 100px - 100px);
    background-color: greenyellow;
  }
}
